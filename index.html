What is this?

A simple passphrase generator that uses the Diceware method to generate secure passphrases.

Why is it collecting my mouse movements?

The app is using the seedrandom.js library to strengthen the pseudo-random number generator by seeding it with mouse coordinates and other sources of entropy. If you use a modern browser, it will use the much more secure window.crypto as its random number source.

Want to have your own?

The complete code for the app is available on GitHub ready to be forked and hosted by yourself.
